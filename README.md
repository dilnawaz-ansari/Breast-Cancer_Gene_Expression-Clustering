# Breast Cancer Gene Expression Clustering

## Overview
Machine learning-based transcriptomic analysis of breast cancer samples using the METABRIC dataset to identify molecular heterogeneity and expression-based patient clusters.

Academic research project focused on cancer genomics and biomarker discovery.

## Dataset
- Source: METABRIC Dataset
- Condition: Breast Cancer Gene Expression Profiles
- Type: Transcriptomic (mRNA Expression) Data
- Samples: ~1900 Patients

## Methods
- Data Preprocessing & Cleaning
- Missing Value Imputation
- Data Normalization (StandardScaler)
- Principal Component Analysis (PCA)
- K-Means Clustering
- Elbow Method for Cluster Selection
- Heatmap Visualization

## Key Results
- Identified 3 distinct molecular clusters
- Reduced dimensionality using PCA
- Detected key genes contributing to cluster separation
- Generated cluster visualization and heatmaps
- Revealed transcriptomic heterogeneity among breast cancer samples

## Important Genes
HSD17B11 | TGFBR2 | KMT2C | EIF4E | SETD1A
JAK1 | MAP2K2 | ITGAY | TNK2 | BAP1
